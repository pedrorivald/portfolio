# 1ª Competição de Projetos: Portfólio Criativo

## Equipe: Vai na Sort

### Executar Projeto:

~~~bash
git clone https://github.com/pedrorivald/portfolio.git
~~~

* > Navegue pelo gerenciador de arquivos até a pasta portfolio

* > Clique duas vezes no arquivo index.html
